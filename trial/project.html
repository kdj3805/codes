MedChat - Healthcare RAG Assistant

AI-powered system to help cancer patients understand clinical reports, treatments, and dietary guidance using medical literature and knowledge graphs.

1. Overview

Cancer patients struggle to understand complex medical reports and fragmented healthcare information. This system bridges that gap using AI-driven retrieval and explanation.

2. System Architecture
3. Pipeline
  • Upload patient report (PDF/TXT)
  • Extract text (Docling / PyMuPDF)
  • Chunk + classify content
  • Generate embeddings
  • Store in Qdrant
  • Hybrid retrieval
  • RRF + MMR
  • LLM response generation
  • Render images
4. Key Features
Multimodal RAG Image Grounding Hybrid Retrieval Auto Analysis
  • PDF ingestion with OCR
  • Image extraction + filtering
  • Caption-based retrieval
  • Patient-friendly explanations
5. Tech Stack
  • Frontend: Streamlit
  • Backend: Python
  • Embeddings: BGE
  • Vector DB: Qdrant
  • LLM: LLaMA 3.3 (Groq)
  • Retrieval: BM25 + RRF + MMR
6. Use Case

Patient uploads report → system explains diagnosis, treatment, diet, and shows relevant medical images.
